PsychlopsJS

About This Project

Psychlops is a free toolkit/library for drawing various types of visual stimulus used in the vision science. This is the JavaScript version running on web-browsers. Web-site of project including C++ version is at [osdn.jp](http://psychlops.osdn.jp/).

Getting Started

Psychlops JS is working on a web-server. Try functionality of the toolkit on the GitHub server.
